  - title: 多环境运行
    details: Ipare 既可以运行在 serverless 也可以运行在原生nodejs服务、微服务等场景。Ipare 原生支持 serverless，作为云函数比其他nodejs框架速度更快。
  - title: 先进特性
    details: Ipare 源码全部使用 TypeScript 开发，有完善的智能提示。mva 架构让项目各模块耦合度更低。可选的装饰器功能让接口设计更加简单。
